# Landing Page - Yasmin Assis

Landing page institucional desenvolvida para apresentar o posicionamento da marca Yas Assis, com foco em estratégia digital, clareza de comunicação e conversão por contato no WhatsApp.

## Visão geral

Este projeto foi construído como uma página única (one page), com seções estratégicas para:

- apresentar identidade e proposta da marca;
- destacar entregas e método de trabalho;
- reforçar tom de voz e público-alvo;
- captar leads por meio de formulário com redirecionamento para WhatsApp.

## Objetivos da página

- fortalecer a percepção de autoridade da marca;
- facilitar a leitura com hierarquia visual clara;
- conduzir a pessoa visitante até o contato;
- manter boa experiência em desktop e mobile.

## Tecnologias utilizadas

- HTML5
- CSS3 (estilização responsiva com media queries)
- JavaScript (interações visuais e formulário de contato)

## Estrutura do projeto

- index.html: arquivo principal com marcação, estilos e scripts da landing page.

## Funcionalidades implementadas

- navegação fixa com destaque de seção ativa;
- animações de entrada ao rolar a página;
- menu adaptado para dispositivos móveis;
- seção de método com cards responsivos;
- formulário de contato no bloco final com envio para WhatsApp.

## Como executar localmente

Opção 1 (Python):

1. Abra o terminal na pasta do projeto.
2. Execute:
   py -m http.server 3000
3. Acesse no navegador:
   http://localhost:3000

Opção 2 (Node.js com Serve):

1. Instale o Serve, caso necessário:
   npm install -g serve
2. Execute na pasta do projeto:
   serve -l 3000
3. Acesse no navegador:
   http://localhost:3000

## Personalização rápida

Para adaptar a landing page para outra marca:

- altere os textos institucionais e chamadas de ação;
- atualize cores e tipografia no bloco de variáveis CSS;
- ajuste links de navegação e contato;
- configure o texto padrão enviado ao WhatsApp no script final.

## Boas práticas recomendadas

- validar contraste de cores para acessibilidade;
- revisar ortografia e consistência de tom de voz a cada atualização;
- testar em diferentes larguras de tela antes de publicar;
- manter o conteúdo objetivo, com foco em conversão.

## Próximos passos sugeridos

- integrar o formulário a uma ferramenta de CRM;
- adicionar métricas (Google Analytics ou similar);
- otimizar SEO on-page com metatags e Open Graph;
- publicar em hospedagem estática (Vercel, Netlify ou GitHub Pages).
